## ğŸ› ï¸ How to Run the Project

Follow these steps to get the ECO CART project up and running on your local machine.

### This project consists of three main modules:
1. **eco-cart (React Native App)**
2. **web_frontend (React App)**
3. **server (Python FastAPI App)**

Follow the steps below to set up and run each module.

---

## 1. **eco-cart (React Native App)**

To run the mobile app, follow these steps:

### Prerequisites
- **Install Node.js and npm**: [Download from Node.js](https://nodejs.org/en/download/)

### Steps to Run
1. **Install Expo CLI** globally:
   ```bash
   npm install -g expo-cli
   ```

2. **Clone the Repository**:
   ```bash
   git clone https://github.com/your_username/eco-cart.git
   ```

3. **Navigate to the `eco-cart` Directory**:
   ```bash
   cd eco-cart
   ```

4. **Install the Dependencies**:
   ```bash
   npm install
   ```

5. **Start the App**:
   ```bash
   npm start
   # or
   expo start
   ```

6. **Scan the QR Code** with the Expo app on your phone or use an emulator/simulator to see the app in action.

---

## 3. **server (Python FastAPI App)**

To run the backend server, follow these steps:

### Prerequisites
- **Install Python 3.7+ and pip**: [Download Python](https://www.python.org/downloads/)

### Steps to Run
1. **Clone the Repository**:
   ```bash
   git clone https://github.com/your_username/eco-cart.git
   ```

2. **Navigate to the `server` Directory**:
   ```bash
   cd server
   ```

3. **Create a Virtual Environment**:
   ```bash
   python3 -m venv venv
   ```

4. **Activate the Virtual Environment**:
   - On Mac/Linux:
     ```bash
     source venv/bin/activate
     ```
   - On Windows:
     ```bash
     venv\Scripts\activate
     ```

5. **Install the Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

6. **Run the Server**:
   ```bash
   uvicorn main:app --host 0.0.0.0 --port 8000
   ```
